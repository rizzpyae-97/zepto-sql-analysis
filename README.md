# Zepto E-Commerce SQL Analysis

## Project Overview
This project analyzes Zepto e-commerce data to extract business insights related to pricing, discounts, and inventory using PostgreSQL.

## Tools Used
- PostgreSQL
- SQL (CTE, Joins, Window Functions, Subqueries)

## Key Business Questions
- Which products offer the highest discounts?
- Which categories generate the most revenue?
- What are the best value products by weight?
- How do discounts vary across categories?

## Dataset
The dataset contains product-level information including price, discounts, stock status, and weight.
